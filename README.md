# 🦷 Dental Landing — AI-лендинг стоматологической клиники

Демо-проект для портфолио. Адаптируется под любую нишу.

**🔗 Live demo:** https://tonib6348-droid.github.io/dental_landing_demo/

## Возможности

- Анимированный лендинг с описанием услуг и ценами
- Форма записи с отправкой заявки в Telegram
- AI-чатбот на базе Groq/Llama (отвечает на вопросы о клинике)
- Прокси через Cloudflare Worker — API-ключ скрыт от пользователей
- Адаптивный дизайн для мобильных устройств

## Технологии

| Компонент | Технология |
|-----------|-----------|
| Frontend | HTML, CSS, JavaScript (без фреймворков) |
| AI-чатбот | Groq API (Llama 3.1), бесплатный тариф |
| Прокси | Cloudflare Workers |
| Уведомления | Telegram Bot API |
| Хостинг | GitHub Pages |

## Быстрый старт

1. Клонируй репозиторий
2. В `index.html` замени `PROXY_URL` на свой Cloudflare Worker URL
3. Замени телефон, название клиники и системный промпт под клиента

> **Cloudflare Worker** (серверная часть для AI-чатбота) предоставляется отдельно при заказе проекта.

## Адаптация под другой бизнес

Все тексты, цены и контакты вынесены в понятные блоки в начале файлов.
Системный промпт чатбота (`SYSTEM_PROMPT` в `index.html`) — меняется под любую нишу за 5 минут.
